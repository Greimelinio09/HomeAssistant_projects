# 🏠 Home Assistant Add-ons & Extras

A collection of custom scripts, automations, blueprints, and dashboards to take your Home Assistant experience to the next level.

## 🚀 Repository Content

This repository contains various resources developed for my own smart home installation:

* **📂 Blueprints:** Reusable automation templates for lighting, security, and more.
* **🤖 Automations:** Advanced YAML automations that go beyond the standard editor.
* **📊 Dashboards:** Inspiration and code snippets for Lovelace (LUI) dashboards.
* **🛠️ Scripts:** Helpful Python or shell scripts for system maintenance.

---

## 🛠️ Installation

Depending on which part of the repo you want to use, follow these steps:

### Adding Blueprints
1. Copy the link to the desired `.yaml` file in the `/blueprints` folder.
2. In Home Assistant, go to **Settings** > **Automations & Scenes** > **Blueprints**.
3. Click on **Import Blueprint** and paste the URL.

### Manual Configuration (YAML)
Copy the relevant code blocks into your `configuration.yaml` or the respective sub-files (e.g., `automations.yaml`). **Note:** Always check and adjust the entity IDs to match your specific system before restarting.
