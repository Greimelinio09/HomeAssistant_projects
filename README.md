# 🐍 Raspberry Pi MQTT Scripts for Home Assistant

This repository contains Python scripts for Raspberry Pi systems that collect sensor data and publish it to Home Assistant via MQTT.

The projects are modular and designed to be easily adaptable and extendable.

---

## 📦 Contents

- 📡 MQTT communication with Home Assistant
- 📊 Sensor data processing (e.g., temperature, distance, GPIO)
- ⚙️ Simple Python scripts for various use cases
- 🔌 Optimized for Raspberry Pi

---

## 🧰 Requirements

- Raspberry Pi (any model)
- Python 3
- MQTT Broker (e.g., Mosquitto)
- Home Assistant
